<h2>
Usage
</h2>

Clone the repository:
```sh
git clone git@github.com:thaisnishimoto/42sp_Webserv.git
```
Compile:
```sh
make
```
Run program:
```
./webserv /path/to/config/file.conf
```
End program: <br> <br>
`crtl` + `c`
